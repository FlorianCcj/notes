Git - Tricks
############

git config
**********

git config core.filemode false
git config --global core.filemode false

git diff
********

git diff --staged
git diff --color-words
git diff --color-words=.

git rebase
**********

git rebase <branch>
git rebase -i <branch>

git remote
**********

git remote -v: print remote origin

git log
*******

git log --graph --oneline --decorate
