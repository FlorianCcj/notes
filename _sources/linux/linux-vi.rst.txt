Linux - Vim
###########

Key
***

* :code:`R`: redo
* :code:`U`: cancel

Sort a part of a file
*********************

* :code:`V`: go in visual mode
* :code:`up and down`: to select the part to sort
* :code:`:sort`: to sort :v:
