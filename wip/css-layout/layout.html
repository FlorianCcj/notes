general

la balise display: * block: commence sur une nouvelle ligne et s etire autant qu il peut sur la gauche et la droite * inline: encadre juste l element position: * static: default,dit non positionne * relative: as static if donc add nothing, add top, right, left, bottom to move it * fixed: fixed in the browser window * absolute: as fixed but in parent

Exemple position

``` .container { position: relative; } nav { position: absolute; left: 0px; width: 200px; } section { /* position is static by default */ // to let enougth place to nav margin-left: 200px; } footer { position: fixed; bottom: 0; left: 0; height: 70px; background-color: white; width: 100%; } body { // to let enougth place to footer margin-bottom: 120px; } ```

Example float

``` nav { float: left; width: 200px; } section { margin-left: 200px; } nav { float: left; width: 25%; } section { margin-left: 25%; } ```

Media queries

@media screen and (min-width:600px) { nav { float: left; width: 25%; } section { margin-left: 25%; } } @media screen and (max-width:599px) { nav li { display: inline; } }

Cards with inline block and clear

``` .box { float: left; width: 200px; height: 100px; margin: 1em; } .after-box { clear: left; } ```

same inline block only

``` .box2 { display: inline-block; width: 200px; height: 100px; margin: 1em; } ```
