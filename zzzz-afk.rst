Elevation
=========

elevation
v+  = 6b+
j   = v+ + y  => y = 2v+
j+  = j + v+  => y = 0
r   = j+ + y  => y = 1j+ = 1j + 1v+ = 4v+
r+  = r + y   => y = j+ = 4v+

equivalence
j+ => 4v+

cout
v+    -> r+ === 10v+
j/j+  -> r+ === 8v+
r     -> r+ === 4v+

Porteur de lumiere  1v+ + 0j+ + 0r = 10 + 00 + 00 = 10-10 = 00
Barbare             1v+ + 3j+ + 0r = 10 + 24 + 00 = 34-07 = 29
Enfant de la foret  0v+ + 0j+ + 1r = 00 + 00 + 04 = 04-03 = 01
Mort Vivant         0v+ + 0j+ + 1r = 00 + 00 + 04 = 04-02 = 02

nombre de carte pure
v+ => 2v
j => 2 * v+ = 4v
j+ => j + v+ = 6v
r => j+ = 6v ou 8v
r+ => r = 6v ou 10v
0e => r + 2 * v+ = 10v ou 14v
1e => 0e + v+ = 12v ou 16v
2e => 1e + v+ = 14v ou 18v
3e => 2e + v+ = 16v ou 20v
4e => 3e + v+ = 18v ou 22v
