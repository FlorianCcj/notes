Elevation
=========

elevation
v+  = 6b+
j   = v+ + y  => y = 2v+
j+  = j + v+  => y = 0
r   = j+ + y  => y = 1j+ = 1j + 1v+ = 4v+
r+  = r + y   => y = j+ = 4v+

v+    -> r+ === 10v+
j/j+  -> r+ === 8v+
r     -> r+ === 4v+

Porteur de lumiere  4v+ + 2j+ + 0r = 40 + 16 + 00 = 56-20 = 36
Barbare             3v+ + 1j+ + 0r = 30 + 08 + 00 = 38-10 = 28
Enfant de la foret  3v+ + 1j+ + 0r = 30 + 08 + 00 = 36-12 = 24
Mort Vivant         5v+ + 2j+ + 0r = 50 + 16 + 00 = 66-11 = 55
